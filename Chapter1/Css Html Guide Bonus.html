Your Friendly HTML & CSS Quick-Reference Guide

Hey there! This guide will walk you through some essential HTML tags and CSS properties you'll use often when building websites. Let’s dive in!

HTML Tags: The Building Blocks

Headings: <h1> - <h6>

Headings are super important because they help structure your content. Think of them as chapter titles in a book. <h1> is the biggest, and <h6> is the smallest. Keep them organized to make your content easy to follow!

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Smaller Heading</h3>

Paragraph: <p>

This is your basic block of text. Whenever you need a paragraph, just wrap it in <p> tags!

<p>This is a paragraph.</p>

Anchor: <a>

Anchors are your links — they let you create hyperlinks to other pages or sections. Just use the <a> tag with a href attribute to link to any URL.

<a href="https://www.example.com">Visit Example</a>

Image: <img>

Need to add an image to your webpage? Use the <img> tag! Just make sure to add the "src" attribute to tell the browser where the image is located and the "alt" attribute for accessibility.

<img src="image.jpg" alt="Description of image">

Lists: <ul>, <ol>, <li>

Use <ul> for unordered (bulleted) lists and <ol> for ordered (numbered) lists. Then, each item in the list goes inside an <li> tag.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
<ol>
    <li>First</li>
    <li>Second</li>
</ol>

CSS Properties: Styling Your Web Pages

Color:

Color is an easy way to make your text pop! You can set the color of any text by using the color property in your CSS. Experiment with different colors to match your website's vibe.

p {
    color: blue;}

Background-color:

The background-color property lets you set the background color of an element. You can change the background of your entire page, or just a specific section.

body {
    background-color: #f4f4f4;}

Font-size:

Font size is key to making your text readable. The font-size property helps you set the size of your text. Play around with different sizes to see what works best for your design.

h1 {
    font-size: 24px;}

Margin:

Margins create space around elements, pushing them away from other elements. It’s like giving your content some breathing room!

div {
    margin: 20px;}

Padding:

Padding is the space inside an element, between the content and the border. Use it to make sure your content isn’t too squished!

p {
    padding: 10px;}

Border:

Borders are great for defining the edges of elements. You can customize the thickness, style, and color of the border to make things stand out.

div {
    border: 2px solid black;}

Width:

The width property lets you control the width of an element. Use it to make your elements fit perfectly within your layout!

div {
    width: 100%;}

Height:

Just like width, the height property helps control the height of an element. Perfect for making sure everything fits neatly on the page.

div {
    height: 200px;}

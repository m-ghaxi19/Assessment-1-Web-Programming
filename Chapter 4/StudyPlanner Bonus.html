Weekly Study Topics

Week 1: HTML & Structure

Learn the basics of HTML, semantic elements, and structuring a webpage.

Week 2: CSS & Styling

Understand how to style webpages using CSS, colors, typography, and layouts.

Week 3: Flexbox & Grid

Explore modern layout techniques like Flexbox and CSS Grid for responsive design.

Week 4: JavaScript Basics

Dive into JavaScript programming, variables, functions, and the DOM.

Week 5: Advanced JavaScript

Cover ES6+ features, event handling, and API integration.
